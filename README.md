# multiagenttrading
tencenthackathon
This paper presents a multi-agent system architecture for automated investment strategy execution, developed and demonstrated at the Tencent Mini Hackathon in Shanghai using the Tencent Hunyuan foundational model. We design a three-agent collaborative framework in which specialised agents handle macro-economic analysis, fundamental stock screening, and investment intent classication respectively. Due to time constraints and ethical consid- erations, only the macroeconomic analysis agent and the fundamental stock screening agent were fully implemented and evaluated.
During testing we observed declining instruction-following delity and shifting safety parameters across multiple prompt rounds, indicating a lack of consistent contextual memory in the base model. This nding aligns with the broader literature on large language model (LLM) alignment: autoregressive models exhibit a well-documented tendency to prioritise user preference over developer predefined rules in the absence of explicit reinforcement mechanisms.
The system targets the China A-share market, where macroeconomic policy and event- driven strategies play an outsized role relative to developed markets. Existing literature on LLM-assisted portfolio management predominantly studies models applied to Nasdaq-listed equities; this paper contributes an empirical evaluation of an open-source LLM in a policy sensitive emerging market setting. Our architecture implements a hierarchical multi-agent communication protocol in which a macro-analysis agent acts as master coordinator, conditionally activating downstream agents based on market regime classication. We demonstrate how cloud-native design patternsincluding microservices, event-driven workows, and containerised deploymentenable the integration of heterogeneous analytical modules into a cohesive investment system. We further provide a Monte Carlo simulation of a semiconductor portfolio constructed by the system and discuss three principal limitations: sequential agent execution, insucient server-side data sources, and the inability of the LLM to maintain workow continuity under operational standards required for production nancial applications.
Keywords: Multi-agent systems, large language models, automated investment strategy, China A-share market, cloud-native architecture, Tencent Hunyuan, portfolio construction.
